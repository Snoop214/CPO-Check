# CPO (Cost Per Order)

Store-level Cost Per Order tracking from attendance/orders data.

## Architecture
- **Google Apps Script** web app
- `Code.gs` — backend logic (data, calculations, auth)
- `Dashboard.html` — single-page frontend with embedded CSS/JS

## Key Features
- Store-level CPO calculation from attendance and order data
- Loaded CPO with champion/supervisor cost allocation
- Role-based dashboard views

## Rules
- DO NOT change data structure or arrangement unless explicitly asked
- Auto-fill admin login credentials on login page
