Process Overview

Cost Per Order (CPO)

Store-level cost tracking — how it works

Hi Team,

The Cost Per Order (CPO) dashboard is now live. This tool provides real-time visibility into how much each order costs to fulfill at the store level — combining attendance data with order volumes to calculate the true cost per order for every store, every day.

The dashboard supports both Basic CPO (picker cost only) and Loaded CPO (including champion and supervisor cost allocation), with built-in budget variance tracking and month-over-month comparison. Automated weekly and monthly reports are sent directly to management.

Below is the complete process overview covering what CPO is, how it’s calculated, the data sources, and key features available in the dashboard.

What Is CPO?

CPO stands for Cost Per Order — a metric that shows exactly how much it costs to fulfill each order at a given store. It is calculated by dividing the total picker cost (based on attendance and vendor rates) by the number of successful orders for that store in a given period.

Formula

Picker Cost ÷ Successful Orders = CPO

How It Works — Step by Step

1
 Attendance data pulled from Know app — the system reads daily attendance records to determine how many pickers were present at each store. Automated
2
 Order volumes fetched per store — successful order counts are pulled from the orders data source, matched by Vendor ID. Automated
3
 Picker cost calculated using vendor-specific rates — Present Days × (Vendor Rate ÷ Working Days). Each 3PL vendor (KRH, Transguard, etc.) has its own contracted rate. Automated
4
 Basic CPO calculated per store — Total Picker Cost ÷ Successful Orders. This gives the baseline cost per order at each location. Automated
5
 Loaded CPO adds champion & supervisor costs — Champion monthly cost (AED 6,400) is allocated across their managed stores, and Supervisor cost (AED 8,500) is distributed similarly. This gives the true fully-loaded cost per order. Automated
6
 Results displayed on live dashboard — sortable tables, KPI cards, charts, and filters by store/chain/city/period. Budget variance and MoM trends are highlighted automatically. Automated
